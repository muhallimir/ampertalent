🎯 EXECUTION APPROACH: HOW TO ACTUALLY DO IT

Step 1: Create GitHub Repository
────────────────────────────────────────────────────────────────────────────
$ mkdir -p /path/to/ampertalent-clone
$ cd /path/to/ampertalent-clone
$ npx create-next-app@latest . --typescript --app
$ git init
$ git add -A
$ git commit -m "initial commit: Next.js 16 scaffold"


Step 2: For EACH PHASE
────────────────────────────────────────────────────────────────────────────

A. Copy files from  original_repo
   $ cp -r ../ original_repo/app/[relevant-folder] ./app/
   $ cp -r ../ original_repo/lib/[relevant-files] ./lib/
   $ cp -r ../ original_repo/components/[relevant-components] ./components/
   $ cp -r ../ original_repo/__tests__/[relevant-tests] ./__tests__/

B. For each file, make replacements:
   $ grep -r "authorize-net\|s3\|bugsnag\|gohighlevel" . --include="*.ts" --include="*.tsx"
   $ vim [file] # Make necessary adjustments for third-party replacements
   
   Examples:
   - S3 → Supabase: Change import from @aws-sdk to @supabase
   - Authorize.net → Stripe: Change function calls and data structures
   - Bugsnag → Sentry: Change import and capture method names
   - GoHighLevel → HubSpot: Change API endpoint mappings

C. Update branding:
   $ sed -i 's/OriginalRepo/Ampertalent/g' $(find . -type f -name "*.ts" -o -name "*.tsx" -o -name "*.css")
   $ sed -i 's/OriginalRepo\.com/ampertalent-demo.vercel.app/g' $(find . -type f -name "*.ts" -o -name "*.tsx")

D. Test:
   $ npm test -- [phase-related-tests]
   $ npm run build
   $ npm run dev  # Check in browser

E. Commit:
   $ git add -A
   $ git commit -m "Phase [N]: [description]"


Step 3: Deploy (Phase 13)
────────────────────────────────────────────────────────────────────────────
$ git push origin main
$ Connect to Vercel in dashboard
$ Set env vars
$ Deploy: Vercel deploys automatically on push
$ Capture STRIPE_WEBHOOK_SECRET from Stripe dashboard
$ Add to Vercel env vars
$ Redeploy


Step 4: Polish (Phase 14)
────────────────────────────────────────────────────────────────────────────
$ npm run lighthouse
$ npm run type-check
$ npm run lint --fix
$ Commit final updates
$ Deploy
